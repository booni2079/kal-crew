점검 시 유의사항

  1. 1. 점검자는 점검 (평가) 전 전체 점검항목을 미리 Review한 후 점검한다.
  2. 2. 점검사실이 주변승객에게 노출되지 않도록 최대한 주의한다.
  3. 3. 점검항목은 5개 Main Category와 각 Main Category별 Sub-category로 구성되어 있다.
    • - 안전/보안, 승무원 서비스 평가, 기내 쾌적성, 기내식, 기내방송
    • - 안전/보안의 경우 비행단계별로 구성되어 있으며, 이외 Category는 주제별로 구성
  4. 4. 점검결과는 객관식 및 주관식으로 평가한다.
    • - 객관식 : 전 항목 필수 작성
      • • Pass : 규정 준수, 만족, 우수함
      • • Fail : 규정 미준수, 미흡, 불만족, 개선 필요
        *Fail의 경우, 하단 세부 항목 중 Fail 사유에 해당되는 부분에 체크 필요
    • - 주관식 (Comments) : 가능한 전 항목에 대해 작성하며, 객관식 평가가 ‘Fail’인 경우, 향후 개선 필요사항 또는 현장에서 즉각 시정 조치한 사항에 대해 세부적으로 명기
  5. 5. 관찰이 불가한 항목의 경우 Comments란에 “-“ 별도 표기
  6. 6. 점검항목에 포함되지 않은 개선 필요사항에 대해서는 6. 기타에 명기하도록 한다.
